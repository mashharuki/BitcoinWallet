# BitcoinWallet
Bitcoinウォレット機能を実装したアプリのリポジトリです。

### 実装予定機能
  1. トランザクション送信機能
  2. 残高照会機能
  3. アカウント作成＆管理機能
  4. トランザクション履歴照会機能

### 利用するAPI
    BitGo社のAPIを利用するので、アカウントの作成とアクセストークンを作成してください。
